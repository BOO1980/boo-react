State

  • React components has a built-in state object
  • The state object is where you store property values belongs to the component
  • When the state object changes, the components re-renders
  • The state object is initialised in the constructor
  • The State object can contain as many properties as you like
  • You can refer to the state object anywhere in the component by using the this.state.propertyName syntax
  • To change a value in the state object, use the this.setState() method
  • When a value in the state object changes, the component will re-render, meaning that the output will change to the new value(s)
  • Always use the setState() method to change the state object, it will ensure that the component knows its been updated and calls the render() method (and all the other lifecycle methods).
