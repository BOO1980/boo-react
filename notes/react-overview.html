React Overview

  • React is a javaScript Library for building user interfaces
  • With react you can design simple views for your state in your application
  • React will efficiently update and render just the right components when your data changes
  • Declarative views make your code more predictable and easier to debug
  • React lets you to build encapsulated components that manage their own state, then composes them to make complex UIs
  • Since component logic is written in JavaScript instead of templates, you can easily pass rich data through your app and keep state of the DOM
  • You can develop new features in react without rewriting existing code
  • React can also render on the server using Node and power mobile apps using react native
  • You can add React to an HTML page in one minute, you can then either gradually expand its presence or keep it contained in a few dynamic widgets
  • You will need to use other libraries with react as its not a complete solution
  • We will tell react what we want and react will build user interfaces on our behalf
  • React is declarative
  • React uses a virtual browsers which contains virtual DOMs
  • React relies on JavaScript
  • React Native allows you to build native mobile applications
